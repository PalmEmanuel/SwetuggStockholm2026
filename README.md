# Learning How to Drink Better Coffee - Swetugg Stockholm 2026

This is the repo for the presentation "Learning How to Drink Better Coffee" at Swetugg Stockholm 2026.

## Brew Recipe of 1 Liter

Approximately 4-6 cups of coffee. Can easily be adapted to for example 500ml with 100ml pours and 30g coffee instead, but a finer grind size.

- 60g coffee / 940ml water
- Brewer - Hario V60 03
- Coarse grind
- 5 Pours
- 92°C water
- 4~5min

1. Reset scale
2. Grind and add coffee (60g)
3. Pour 140g (200g)
4. Bloom 35~ seconds
5. Repeat 4 times:
   Pour 200g
   Wait 10~ seconds
